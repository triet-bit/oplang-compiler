OPLang Programming Language
Version 1.0 · August 2025 · Ho Chi Minh City University of Technology
OPLang (O-P-Lang) is a mini object-oriented programming language designed for students practicing implementing a compiler for a simple OOP language.
Despite its simplicity, OPLang includes most important features of an OOP language: encapsulation, information hiding, class hierarchy, inheritance, and polymorphism.
Compiler Pipeline
This IDE compiles OPLang source code through the following pipeline:
Source Code → ANTLR4 Lexer / Parser → Abstract Syntax Tree (AST) → Static Semantic Checker → JVM Bytecode (via Jasmin assembler) → Execution on JVM
Quick Start
class Hello { void main() { io.writeStrLn("Hello, OPLang!"); } }
Program Structure
An OPLang program is written in a single file and consists of multiple class declarations. The entry point is a void main() static method.
Class Declaration
class ClassName extends SuperClass { // attributes and methods }
Members can be static or instance. Attributes can be final (immutable) or mutable.
Attribute Declaration
static final int MAX := 100; float length, width; int x := 0, y := 5;
Method Declaration
ReturnType methodName(Type param1; Type param2) { // body } // Reference return Type & methodName(params) { ... }
Constructor & Destructor
// Default constructor ClassName() { ... } // Copy constructor ClassName(ClassName other) { ... } // User-defined constructor ClassName(Type param1; Type param2) { ... } // Destructor ~ClassName() { ... }

Identifiers
Begin with a letter (A-Z, a-z) or underscore, may contain letters, underscores, and digits. OPLang is case-sensitive.

Array Types
int[5] arr := {1, 2, 3, 4, 5}; float[3] coords; arr[0] := 10; # indexing
Reference Types
int & ref := someVar; # reference variable Type & methodName(...) { } # reference return type void method(int & param) # reference parameter
Type Coercion
int is automatically coerced to float when needed. No implicit coercion in the other direction.
Scope Rules
Class scope — instance attributes visible in all methods and subclasses.
Method scope — parameters/variables declared in method body.
Block scope — variables declared in a block, visible to end of that block.
Statements
Assignment
x := 5; this.length := 10.0;
If Statement
if condition then statement; if condition then statement; else statement; if condition then { block } else { block }
For Loop
for i := 1 to 10 do { io.writeIntLn(i); } for i := 10 downto 1 do { io.writeIntLn(i); }
While / Do-While
while condition do { // body } do { // body } while condition;
Return, Break, Continue
return value; # return with value return; # void return break; # exit loop continue; # next iteration
Object Creation
ClassName obj := new ClassName(); ClassName obj := new ClassName(arg1, arg2);

Example Programs
Example 1 — Factorial
class Example1 { int factorial(int n) { if n == 0 then return 1; else return n * this.factorial(n - 1); } void main() { int x; x := io.readInt(); io.writeIntLn(this.factorial(x)); } }
Example 2 — Inheritance & Polymorphism
class Shape { float length, width; float getArea() {} Shape(float length, width) { this.length := length; this.width := width; } } class Rectangle extends Shape { float getArea() { return this.length * this.width; } } class Triangle extends Shape { float getArea() { return this.length * this.width / 2; } } class Example2 { void main() { Shape s; s := new Rectangle(3, 4); io.writeFloatLn(s.getArea()); # 12.0 s := new Triangle(3, 4); io.writeFloatLn(s.getArea()); # 6.0 } }
Example 3 — Constructor & Destructor
class Rectangle { float length, width; static int count; Rectangle() { this.length := 1.0; this.width := 1.0; Rectangle.count := Rectangle.count + 1; } Rectangle(float length; float width) { this.length := length; this.width := width; Rectangle.count := Rectangle.count + 1; } ~Rectangle() { Rectangle.count := Rectangle.count - 1; io.writeStrLn("Rectangle destroyed"); } float getArea() { return this.length * this.width; } static int getCount() { return Rectangle.count; } } class Example3 { void main() { Rectangle r1 := new Rectangle(); Rectangle r2 := new Rectangle(5.0, 3.0); io.writeFloatLn(r1.getArea()); # 1.0 io.writeFloatLn(r2.getArea()); # 15.0 io.writeIntLn(Rectangle.getCount()); # 2 } }
